# 欢迎来到技术文档站 📚

## 关于本站

这是一个个人技术文档站点，用于记录和分享技术学习笔记、项目经验和最佳实践。

## 内容分类

### 🔧 C/C++开发
深入学习C/C++语言特性、内存管理、STL、现代C++等内容。

### 🌐 分布式系统
探索分布式系统的理论与实践，包括：
- **CS架构**：传统的客户端-服务器架构
- **DDS架构**：数据分发服务，去中心化架构
- **微服务**：现代分布式系统架构

### 🔌 网络编程
TCP/IP、Socket编程、HTTP协议、epoll等网络编程核心技术。

### 🐧 Linux开发
Linux系统编程、进程管理、IPC、内核开发等。

### 🚀 Golang
Go语言学习，包括并发编程、channel、goroutine等。

### 💾 数据库
MySQL、Redis、MongoDB等数据库技术。

### 📊 算法
数据结构与算法的学习笔记。

## 如何使用

1. **浏览文档**：点击顶部导航栏选择不同的技术分类
2. **左侧目录**：查看当前分类下的所有文档
3. **右侧导航**：快速跳转到页面的不同章节
4. **搜索功能**：使用右上角搜索框快速查找文档
