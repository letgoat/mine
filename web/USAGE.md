# 五毛仔学技术 - 使用指南

## 快速开始

### 启动网站

```bash
# 进入项目目录
cd web

# 执行启动脚本
./start.sh

# 或者直接使用 Python
python3 -m http.server 8000
```

然后在浏览器中访问：`http://localhost:8000`

## 页面布局说明

### 三栏布局

网站采用三栏布局设计：

```
┌─────────────────────────────────────────────────┐
│          顶部导航栏（固定）                       │
├──────┬─┬──────────────────────┬─┬──────────────┤
│      │ │                      │ │              │
│ 左   │分│    主内容区          │分│  右侧导航    │
│ 侧   │隔│   (Markdown文档)     │隔│  (页面锚点)  │
│ 目   │线│                      │线│              │
│ 录   │ │                      │ │              │
│      │ │                      │ │              │
└──────┴─┴──────────────────────┴─┴──────────────┘
```

### 拖动调整

- **左侧分隔线**：拖动可调整左侧栏宽度（160px ~ 320px）
- **右侧分隔线**：拖动可调整右侧栏宽度（140px ~ 200px）
- 调整后的宽度会自动保存，下次打开保持

### 响应式适配

- **大屏幕**（> 1200px）：显示三栏
- **中等屏幕**（768px ~ 1200px）：只显示左侧栏和主内容
- **小屏幕**（< 768px）：单栏，点击菜单按钮切换侧边栏

## 功能使用

### 1. 搜索功能

- 点击顶部搜索框或按 `Ctrl + K`
- 输入关键词实时过滤左侧文档列表

### 2. 导航使用

**顶部导航**：
- 点击分类名称切换不同专栏
- 激活的分类会高亮显示（橙色背景）

**左侧目录**：
- 按章节分组显示文档列表
- 点击文档名称加载内容
- 当前文档会高亮显示

**右侧导航**：
- 自动提取文档中的标题（h2, h3, h4）
- 点击标题快速跳转到对应位置
- 当前章节会高亮显示

### 3. 清除缓存

如果页面布局显示异常：

**方法1**：访问清除缓存页面
```
http://localhost:8000/clear-cache.html
```

**方法2**：手动清除
1. 按 `F12` 打开开发者工具
2. 切换到 Console 标签
3. 执行命令：
```javascript
localStorage.clear();
location.reload();
```

**方法3**：浏览器清除
- 按 `Ctrl + Shift + Delete`
- 选择"Cookies和其他网站数据"
- 清除数据

## 编写文档

### 文档格式

所有文档使用 Markdown 格式（`.md` 文件）

### 创建新文档

1. 在对应分类的 `docs/` 子目录下创建 `.md` 文件
2. 文件名使用小写字母和连字符（例如：`smart-pointer.md`）
3. 使用标准 Markdown 语法编写内容

### 文档模板


## 配置修改

### 添加新分类

1. **修改 `js/config.js`**：
```javascript
categories: {
    newCategory: {
        name: '新分类名称',
        sections: [
            {
                title: '章节名称',
                docs: [
                    { title: '文档标题', file: 'newcategory/doc1.md' }
                ]
            }
        ]
    }
}
```

2. **修改 `index.html`**，在导航栏添加：
```html
<a href="#" class="nav-item" data-category="newCategory">新分类名称</a>
```

3. **创建文档目录**：
```bash
mkdir -p docs/newcategory
```

### 修改主题颜色

编辑 `css/style.css`：

```css
:root {
    --primary-color: #ff7700;  /* 主色调（橙色）*/
    --hover-color: #ff9933;    /* 悬停色 */
    --bg-color: #f5f5f5;       /* 背景色 */
}
```

### 修改网站名称

编辑 `index.html`：

1. 修改 `<title>` 标签
2. 修改 `.logo-text` 内容
3. 修改首页欢迎文字

## 故障排除

### 问题1：文档加载失败

**原因**：文档文件不存在或路径错误

**解决**：
1. 检查 `js/config.js` 中的文件路径
2. 确认 `docs/` 目录下有对应文件
3. 查看浏览器控制台（F12）的错误信息

### 问题2：代码不高亮

**原因**：代码块语言标识错误

**解决**：
确保代码块使用正确的语言标识符：
````markdown
```cpp
// 使用 cpp 而不是 c++
```
````

### 问题3：布局错乱

**原因**：浏览器缓存了旧的宽度设置

**解决**：
访问 `http://localhost:8000/clear-cache.html` 清除缓存

### 问题4：搜索不工作

**原因**：JavaScript 错误或文档列表未加载

**解决**：
1. 检查浏览器控制台错误
2. 确认已选择了文档分类
3. 刷新页面重试
